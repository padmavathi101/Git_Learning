Git_commands

• git status;check status and which branch at present
• git checkout -b new_branch_name;to create a new branch locally,-b option to create a new branch
• git push --set-upstream origin new_branch_name; push branch changes to remote
• git add File_name_modified_or_added; to move files to staging area
• git reset HEAD File_name_modified_or_added;to unstage
• git commit -m "File_name_modified_or_added";adding staged changes to commit locally
• git push origin firstbranch;push commited changes to server
• git checkout masteroranyotherbranch;go to that branch
• git merge anybranch;merge anybranch to present checkedout branch(let's say master).
